2 Install

2.1 Ubuntu

$ sudo apt-get install git

2.2 CentOS

$ sudo yum install git

2.3 Fedora

$ sudo yum install git-core

2.4 Arch Linux

$ sudo pacman -Sy git

2.5 Gentoo

$ sudo emerge --ask --verbose dev-vcs/git

2.6 macOS

$ brew install git
