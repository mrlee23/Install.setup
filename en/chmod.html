Overview

Change the permission of file or directory.

How to Install

Install

This is builtin command.

Usage

Syntax

chmod [OPTION]... MODE[,MODE]... FILE...)}}}
chmod [OPTION]... OCTAL-MODE FILE...)}}}    
chmod [OPTION]... --reference=RFILE FILE...)

Description

Reference

Reference Class Description
u Use Owner of file
g Group Group member of file
o Others  
a All Same with "ugo"

Operator

Operator Description
+ Add modes to exists class
- Remove modes from exists class
= Set modes

8 Bits mode

# Permission rwx bin
0 none --- 000
1 execute only --x 001
2 write only -w- 010
3 write and execute -wx 011
4 read only r-- 100
5 read and execute r-x 101
6 read and write rw- 110
7 read, write and execute rwx 111

Symbol mode

mode name description
r read Read file or access directory list.
w write Write file or directory.
x execute Execute.
X special execute  
s setuid/gid  
t sticky bit
