** 옵션
   :PROPERTIES: 
   :CUSTOM_ID: options
   :LANG: ko
   :END:      

- {{{option(--normal)}}} :
  (기본값) 일반적인 출력으로 보여줍니다.
- {{{option(-q)}}}, {{{option(--brief)}}} :
  파일이 다를 경우만 기록합니다.
- {{{option(-c)}}}, {{{option(-C)}}} {{{argument(NUM)}}}, {{{option(--context)}}} {{{argument([=NUM])}}} :
  복사된 문맥을 {{{arg(NUM)}}} 수(기본 값 3)만 큼 출력합니다.
- {{{option(-u)}}}, {{{option(-U)}}} {{{argument(NUM)}}}, {{{option(--unified)}}} {{{argument([=NUM])}}} :
  통일된 문맥을 {{{arg(NUM)}}} 수(기본 값 3)만 큼 출력합니다.
- {{{option(-e)}}}, {{{option(--ed)}}} :
  ed 스크립트로 출력합니다.
- {{{option(-n)}}}, {{{option(--rcs)}}} :
  RCS 포맷으로 출력합니다.
- {{{option(-y)}}}, {{{option(--side-by-side)}}} :
  2개의 열로 출력합니다.
- {{{option(-W)}}}, {{{option(--width)}}} {{{argument(=NUM)}}} :
  {{{arg(NUM)}}}으로 최대 출력 열 수를 정합니다. (기본값 130)
- {{{option(--left-column)}}} :
  같은 줄의 왼쪽 열만 출력합니다.
- {{{option(--supress-common-lines)}}} :
  같은 줄은 출력하지 않습니다.
- {{{option(-p)}}}, {{{option(--show-c-function)}}} :
  각 차이를 포함하는 C 함수를 출력합니다.
- {{{option(-F)}}}, {{{option(--show-function-line)}}} {{{argument(=RE)}}} :
  {{{arg(RE)}}}와 가장 최근에 매칭된 줄을 출력합니다.
- {{{option(--label)}}} {{{argument(LABEL)}}}
- {{{option(-t)}}}, {{{option(--expand-tabs)}}}
