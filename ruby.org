* Prerequisites
- [[./apt-add-repository.org][apt-add-repository]]

* Install

#+NAME: install_ruby
#+BEGIN_SRC sh
$ sudo apt-add-repository ppa:brightbox/ruby-ng
$ sudo apt-get update
$ sudo apt-get install ruby2.5 ruby-switch
$ sudo ruby-switch --set ruby2.5
$ sudo apt-get install ruby2.5-dev
#+END_SRC
