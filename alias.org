** 소개
  :PROPERTIES: 
  :LANG: ko
  :END:    

명령어를 대신할 별명을 만들고 실행할 수 있도록 만든다.

** 특징
  :PROPERTIES: 
  :LANG: ko
  :END:    

- 쉘 내장 명령어
- 명령어 축약                                                                                                  
- 자주 사용하는 명령어를 줄여서 실행 가능하게 해준다.                                                          
- 실행시 =PATH= 변수의 디렉토리 내 실행파일보다 먼저 실행되므로 만약 cd = clear 로 설정되었을 경우 cd 명령을 치면 clear가 실행된다.

** 구조
   :PROPERTIES: 
   :CUSTOM_ID: syntax
   :LANG: ko
   :END:      

	#+BEGIN_SRC sh
alias [-p] [이름[=값] ...]
	#+END_SRC

** 설정
   :PROPERTIES: 
   :CUSTOM_ID: configuration
   :LANG: ko
   :END:      

{{{file(~/.bash_profile)}}}이나 {{{file(~/.bashrc)}}}에서 alias 설정을 할 수 있습니다.

** 옵션
   :PROPERTIES: 
   :CUSTOM_ID: options
   :LANG: ko
   :END:      

- {{{option(-p)}}}
  - 현재 쉘에 설정된 alias를 모두 보여준다.

** 사용 예
   :PROPERTIES: 
   :CUSTOM_ID: examples
   :LANG: ko
   :END:      

alias로 설정한 이름을 명령어 처럼 사용할 수 있습니다.

	#+BEGIN_SRC sh
$ alias cl='clear'
$ alias ll='ls -l'
$ alias rm='rm -i'
	#+END_SRC

	#+BEGIN_SRC sh
$ ll
	#+END_SRC
