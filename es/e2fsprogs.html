Instalar

Ubuntu

$ sudo apt-get install es2fsprogs

CentOS

$ sudo yum install es2fsprogs

from Source code

$ git clone https://git.kernel.org/pub/scm/fs/ext2/e2fsprogs.git
$ mkdir build
$ cd build
$ ../configure
$ make
$ make check
$ sudo make install
