Cómo instalar

Dependencias

  • libc6 (>= 2.17)
    • GNU C Library: Shared libraries also a virtual package provided by libc6-udeb
  • libsigsegv2 (>= 2.9)
    • Library for handling page faults in a portable way

Instalar

Ubuntu

$ sudo apt-get install m4

CentOS

$ sudo yum install m4
