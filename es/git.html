Instalar

Ubuntu

$ sudo apt-get install git

CentOS

$ sudo yum install git

Fedora

$ sudo yum install git-core

Arch Linux

$ sudo pacman -Sy git

Gentoo

$ sudo emerge --ask --verbose dev-vcs/git

macOS

$ brew install git
