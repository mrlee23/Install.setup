Cómo instalar

Requisitos previos

Ubuntu

  • apt-add-repository

Mac OS

  • brew

Instalar

Ubuntu

Emacs 25
$ sudo apt-add-repository -y ppa:adrozdoff/emacs
$ sudo apt update
$ sudo apt install emacs25

Mac OS

$ brew install emacs
